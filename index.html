how to use focus
⏱ Timer
1️⃣
เริ่ม session เลือก project → พิมพ์ชื่องาน → กด start
⏹
หยุดกลางคัน กด done เพื่อหยุดและบันทึกเวลาจริงที่ทำไป — จากนั้นจะมี slider ให้ rate effort ก่อน (กด skip ได้ถ้าไม่อยาก rate)
🔴
4 Dots — วงจร Pomodoro ทุก 4 sessions → long break อัตโนมัติ dots reset เป็น 0/4 หลัง long break จบ
⚙
ตั้งเวลา กด gear icon → เลือก preset หรือ custom (classic 25/5 · long focus 50/10 · deep work 90/15)
😌
Effort Rating หลัง session จบ เลื่อน slider บอกว่า session หนักแค่ไหน → กด done ✓ บันทึก (break ยังนับต่อ) หรือ skip → เพื่อข้าม break
🏷 Project Tags
👆
เลือก project กด pill → prefix นำหน้า task เช่น "Work · Design" · กดซ้ำ → ยกเลิก
➕
เพิ่ม / ลบ project กด + add → พิมพ์ชื่อ → Enter · Long press pill → ลบ
🎵 Ambient Sounds
🌧
Ambient rain · forest · café · white noise · brown noise — เปิดลำโพงได้เลย
🎧
Tones — เปิดพร้อม Ambient ได้เลย alpha 🎧 binaural 10Hz · 40hz 🎧 gamma · 528hz pure tone · Ambient และ Tone มี volume แยกกัน — เลือกได้อิสระ เปิดพร้อมกันหรืออันเดียวก็ได้ · เสียงที่มี 🎧 ต้องใช้หูฟัง
⚠️
เสียงไม่ออก? แตะหน้าจอก่อน 1 ครั้ง · ตรวจสอบว่าไม่ได้เปิด Silent Mode (สวิตช์ข้างเครื่องเห็นสีส้ม)
✅ Todo
✏️
เพิ่ม / จัดการ พิมพ์ → Enter หรือ + · แตะ ○ → เสร็จแล้วหายอัตโนมัติ · ลาก ⠿ เพื่อเรียงลำดับ
↩️
Carry-over กด clear all → tick task ที่อยากเก็บไว้พรุ่งนี้ → confirm
📝 Notes
⚡
4 ประเภท 📝 General · 💡 Idea · 🅿 Parking lot · ⚡ Action — เลือกก่อน save
↓
Export → Obsidian กด ↓ บน note card → ได้ .md พร้อม frontmatter · export all ↓ รวมทุก note
📊 Stats
📅
Today / This Week เลือก tab ด้านบน · week view แสดง bar chart 7 วัน + breakdown by project
💾 Backup
💾
Export / Import กด ⚙ → เลื่อนลง → ↓ export data เพื่อ backup · ↑ import data เพื่อ restore
⌨️ Keyboard (Desktop)
⌨️
Shortcuts Space start/pause · R reset · N open notes
ใช้งานได้เมื่อไม่มี input field ถูก focus อยู่
☕ Break Overlay
☕
ระหว่าง break break overlay แสดงเต็มจอ — สามารถใช้ nav bar ด้านล่าง สลับไป Notes หรือ Todo ได้เลยโดยไม่ต้อง skip
😌
Effort Rating เลื่อน slider → กด done ✓ บันทึก effort (break นับต่อ) · กด skip break → เพื่อข้าม break ทั้งหมด
🔥 Focus Heatmap
🔥
Peak focus hours ดูใน Stats → This Week — grid 6:00–23:00 สีเข้ม = focus เยอะ วาง งานสำคัญ ในช่วงสีเข้มที่สุด
ข้อมูลสะสมจากทุก sessions — ยิ่งใช้นานยิ่งแม่นยำ
📝 Markdown ใน Notes
✍️
พิมพ์ Markdown ได้เลย # หัวข้อ · * bullet · **bold** · ` code ` — แสดงผลถูกต้องใน view mode
Edit mode และ Export .md ยังเป็น raw markdown เหมือนเดิม
